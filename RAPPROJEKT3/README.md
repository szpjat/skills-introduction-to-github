# RAPPROJEKT3

## Opis projektu
To jest katalog projektu RAPPROJEKT3, w którym możesz wrzucać pliki z pracy w Visual Studio Code.

## Struktura katalogów

- **src/** - Kod źródłowy projektu
- **docs/** - Dokumentacja projektu
- **assets/** - Zasoby projektu (obrazy, pliki statyczne, itp.)

## Jak dodawać pliki

### Przez interfejs GitHub:
1. Otwórz folder RAPPROJEKT3
2. Kliknij "Add file" → "Upload files"
3. Przeciągnij pliki z Visual Studio Code lub kliknij "choose your files"
4. Dodaj opis zmian (commit message)
5. Kliknij "Commit changes"

### Przez Git w Visual Studio Code:
1. Sklonuj to repozytorium:
   ```bash
   git clone https://github.com/szpjat/skills-introduction-to-github.git
   ```

2. Przejdź do katalogu RAPPROJEKT3:
   ```bash
   cd skills-introduction-to-github/RAPPROJEKT3
   ```

3. Skopiuj swoje pliki do odpowiednich folderów (src, docs, assets)

4. Dodaj i zatwierdź zmiany:
   ```bash
   git add .
   git commit -m "Dodanie moich plików z Visual Studio Code"
   git push
   ```

## Gotowe!
Możesz teraz przesyłać swoje pliki z projektów Visual Studio Code do tego katalogu.
